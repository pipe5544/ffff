Atributos válidos

body Define el cuerpo del documento HTML, como la parrte central del sitio, es la parte donde se describe el contenido la página web, su estructura, su forma, sus colores, textos, etc.
head Representa la cabeza del sitio web, es la parte donde se incluye la información acerca del documento a mayoría de esta información no la muestra el navegador al usuario, provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.
footer Indica el pie de página del sitio we, donde se destaca información reelvante como derechos de autor información del sitio, documentos relacionados etc.



Estilos CSS

CSS-Colors Los colores se especifican utilizando nombres de color predefinidos o valores RGB, HEX, HSL, RGBA, HSLA.
CSS Color Names En CSS, se puede especificar un color utilizando un nombre de color, encontramos 140 nombres de colores estándar
    Ejemplo:
  • black
  • blue
  • tomato
  • gray
Valor CSS RGB En CSS, un color se puede especificar como un valor RGB, usando esta fórmula:
rgb (rojo, verde, azul) se la intensidad del color entre 0 y 255.
    Ejemplo:
  • rgb (0,0,0)
  • rgb(255,255,255)
Valor CSS HEX En CSS, se puede especificar un color usando un valor hexadecimal en la forma:
#rrggbb
Donde rr (rojo), gg (verde) y bb (azul) son valores hexadecimales entre 00 y ff (igual que el decimal 0-255).
    Ejemplo:
  • #00ff00
  • #00000
  • #fffff
Valor HSL En CSS, se puede especificar un color usando tono, saturación y luminosidad (HSL) en la forma:
hsl (tono, saturación, luminosidad)
Donde tono va de o a 255, saturación va de forma porcentual %, y iluminación va de forma porcentual %.
    Ejemplo
  • hsl(64, 100%, 50%)
  • hsl(0, 48%, 35%)
Valor RGBA En CSS, Los valores de color RGBA son una extensión de los valores de color RGB con un canal alfa, que especifica la opacidad de un color de la forma:
rgba (rojo, verde, azul, alfa)
El parámetro alfa es un número entre 0.0 (totalmente transparente) y 1.0 (no transparente en absoluto)
    Ejemplo
  • rgba(117, 255, 71, 0.5)
  • rgba(0, 255, 255, 0.5)
CSS-background Las propiedades de background CSS se utilizan para definir los efectos de fondo para los elementos.
CSS background-color La propiedad background-color especifica el color de fondo de un elemento.
Se especifica con mayor frecuencia el color de fondo por:
  • Un nombre de color válido
  • un valor HEX
  • Un valor RGB
CSS background-image La propiedad background-image especifica una imagen para usar como fondo de un elemento, por defecto, la imagen se repite para que cubra todo el elemento.
CSS background-position La propiedad background-position se usa para especificar la posición de la imagen de fondo.
CSS background-attachment La propiedad background-attachment especifica si la imagen de fondo debe desplazarse o ser fija.
CSS-Borders Las propiedades CSS borderle permite especificar el estilo, el ancho y el color del borde de un elemento.
CSS Border Width La propiedad border-width especifica el ancho de los cuatro bordes,se puede establecer un tamaño específico px, pt, cm, em, etc, va representado como las manecillas del relog iniciando desde el borde superior, el borde derecho, el borde inferior y el borde izquierdo.
    Ejemplo
  • border-width: 5px 15px 14px 20px;
CSS Border -Lados individuales En CSS, también hay propiedades para especificar cada uno de los bordes superior, derecho, inferior e izquierdo, (top, right, bottom, and left)
CSS Border – propiedad abreviada Es posible especificar todas las propiedades de borde individuales en una propiedad.
    Ejemplo
  • border: 3px solid black;
Borde izquierdo (Border -Left) Es posible especificar un borde al costado izquierdo con la propiedad border-left
    Ejemplo
  • border-left: 3px solid black;
Borde inferior (Border -Bottom) Es posible especificar un borde inferior con la propiedad border-bottom
    Ejemplo
  • border-bottom: 3px solid red;
Bordes redondeados (border-radius) Es posible especificar un borde redondeado con la propiedad border-radius
    Ejemplo
  • border-radius: 3px;
CSS Margins La propiedad CSS margin se usan para crear espacio alrededor de elementos, fuera de cualquier borde definido, existen propiedades para establecer el margen para cada lado de un elemento tanto arriba, derecha, abajo e izquierda, ( top, right, bottom, and left)
Margin – lados individuales CSS tiene propiedades para especificar el margen para cada lado de un elemento:
  • margin-top arriba
  • margin-right derecha
  • margin-bottom abajo
  • margin-left izquierda

Todas las propiedades de margen pueden tener los siguientes valores:
  • %: especifica un margen en% del ancho del elemento contenedor
  • auto: el navegador calcula el margen
  • Inherit: especifica que el margen debe heredarse del elemento padre.
  • length: especifica un margen en px, pt, cm, etc.
    Ejemplo margenes individuales
  • margin-top: 10px;
  • margin-bottom: 5px;
  • margin-right: 50px;
  • margin-left: 70px;
Margin – propiedad abreviada Para acortar el código, es posible especificar todas las propiedades de margen en una propiedad.
    Ejemplo agrupando el ejemplo anterior
  • margin: 10px 5px 50px 70px;
Margin auto La propiedad de margen en automático para centrar horizontalmente el elemento dentro de su contenedor.
    Ejemplo
  • margin: auto;
El valor inherit (heredado) Es posible que una clase hijo herede la margen de una clase padre
    Ejemplo
  • margin-left: inherit;
CSS Padding La propiedad CSS padding se usan para generar espacio alrededor del contenido de un elemento, dentro de cualquier borde definido, CSS, tienes control total sobre el relleno. Hay propiedades para configurar el relleno para cada lado de un elemento arriba, derecha, abajo e izquierda, (top, right, bottom, and left).
Pading – lados individuales CSS tiene propiedades para especificar el relleno para cada lado de un elemento:
  • margin-top arriba
  • margin-right derecha
  • margin-bottom abajo
  • margin-left izquierda

Todas las propiedades de margen pueden tener los siguientes valores:
  • %: especifica un margen en% del ancho del elemento contenedor
  • auto: el navegador calcula el margen
  • Inherit: especifica que el margen debe heredarse del elemento padre.
  • length: especifica un margen en px, pt, cm, etc.
    Ejemplo margenes individuales
  • pading-top: 10px;
  • pading-bottom: 5px;
  • pading-right: 50px;
  • pading-left: 70px;
Pading – propiedad abreviada Para acortar el código, es posible especificar todas las propiedades de relleno (pading) en una propiedad.
    Ejemplo agrupando el ejemplo anterior
  • pading: 10px 5px 50px 70px;
Padding (box-sizing) Para mantener un ancho establecido, sin importar la cantidad de relleno, puede usar la propiedad debox-sizing. Esto hace que el elemento mantenga su ancho; Si aumenta el relleno, el espacio de contenido disponible disminuirá.
    Ejemplo:
    width: 500px;
    padding: 5px;
    box-sizing: border-box

CSS Fonts Las propiedades de fuente CSS definen la familia de fuentes
CSS Font Families En CSS, hay dos tipos de nombres de familias de fuentes:
  • Generc-family (familia genérica): un grupo de familias de fuentes con un aspecto similar (como "Serif" o "Monospace")
  • font family (familia de fuentes): una familia de fuentes específica (como "Times New Roman" o "Arial")
Font Family La propiedad de font-family debe contener varios nombres de fuentes como un sistema "alternativo". Si el navegador no admite la primera fuente, intenta con la siguiente fuente, y así sucesivamente.
    Ejemplo:
  • font-family: "Times New Roman", Times, serif;
Font Style La propiedad font-style se usa principalmente para especificar texto en cursiva.
    Esta propiedad tiene tres valores:
  • normal : el texto se muestra normalmente
  • italic: el texto se muestra en cursiva
  • oblique: el texto es "inclinado" (oblicuo es muy similar a cursiva, pero menos compatible)
    Ejemplo:
  • font-style: normal;
Font Size La propiedad font-size establece el tamaño del texto.
    Ejemplo:
  • word-spacing: 10px;
Font Size con Pixels Ajustar el tamaño del texto con píxeles le da un control total sobre el tamaño del texto:
    Ejemplo:
  • font-size: 16px;
Font Size con Em Para permitir a los usuarios cambiar el tamaño del texto
    Ejemplo:
  • font-size: 2.5em;
Font Weight La propiedad font-weight especifica el peso de una fuente:
    Ejemplo:
  • font-weight: normal;
Responsive Font Size (Tamaño de fuente sensible) El tamaño del texto se puede establecer con una unidad vw, de esa forma, el tamaño del texto seguirá el tamaño de la ventana del navegador
Font Variant La propiedad font-variant especifica si un texto debe mostrarse o no en letra minúscula, las letras minúsculas se convierten en letras mayúsculas. Sin embargo, las letras mayúsculas convertidas aparecen en un tamaño de fuente más pequeño que las letras mayúsculas originales en el texto.
    Ejemplo:
  • font-variant: small-caps;
CSS links Con CSS, los enlaces o anclas de navegación se pueden diseñar de diferentes maneras.
Styling Links (enlaces de estilo) Los enlaces pueden tener un estilo diferente según el estado en que se encuentren.
    En CSS existen cuatro estados de enlaces son:
  • a:link: un enlace normal, no visitado
  • a:visited: un enlace que el usuario ha visitado
  • a: hover: un enlace cuando el usuario pasa el mouse sobre él
  • a:active: un enlace en el momento en que se hace clic
Botones de enlace (Link Buttons ) Con CSS se pueden modificar los estilos de los enlaces para visualizarlos de forma de botones.

CSS position Propiedad La propiedad position especifica el tipo de método de posicionamiento utilizado para un elemento estático, relativo, fijo, absoluto o fijo. (static, relative, fixed, absolute or sticky).
    Hay cinco valores de posición diferentes:
  • static

Elementos superpuestos Una tabla responsive mCuando los elementos se colocan, pueden superponerse a otros elementos, La propiedad z-index especifica el orden de apilamiento de un elemento (qué elemento debe colocarse delante o detrás de los demás), mostrará una barra de desplazamiento horizontal si la pantalla es demasiado pequeña para mostrar el contenido completo.
CSS Layout - Overflow La propiedad CSS overflow controla lo que sucede con el contenido que es demasiado grande para caber en un área, especifica si se debe recortar el contenido o agregar barras de desplazamiento cuando el contenido de un elemento es demasiado grande para caber en el área especificada.
overflow: visible Por defecto, el overflow es visible, lo que significa que no está recortado y se representa fuera del cuadro del elemento.
overflow: hidden Con el valor oculto, se recorta el desbordamiento y se oculta el resto del contenido.
    Ejemplo:
  • overflow: hidden;
overflow: scroll Al establecer el valor para desplazarse, el overflow se recorta y se agrega una barra de desplazamiento para desplazarse dentro del cuadro. Tenga en cuenta que esto agregará una barra de desplazamiento tanto horizontal como verticalmente (incluso si no la necesita)
    Ejemplo:
  • overflow: scroll;
overflow: auto El valor automático es similar al desplazamiento, pero agrega barras de desplazamiento solo cuando es necesario.
    Ejemplo:
  • overflow: auto;
overflow-x and overflow-y Las propiedades overflow-x y overflow-y especifican si se cambia el desbordamiento de contenido solo horizontal o verticalmente (o ambos).
  • overflow-x especifica qué hacer con los bordes izquierdo / derecho del contenido.
  • overflow-y especifica qué hacer con los bordes superior / inferior del contenido.
    Ejemplo:
  • overflow-x: hidden;
  • overflow-y: scroll;
CSS - float and clear La propiedad float CSS especifica cómo debe flotar un elemento.
  • La propiedad float se usa para posicionar y formatear contenido, e.g. deje que una imagen flote a la izquierda del texto en un contenedor.
  • La propiedad clear CSS especifica qué elementos pueden flotar junto al elemento borrado y de qué lado.
    La propiedad flotante puede tener uno de los siguientes valores:
  • left: el elemento flota a la izquierda de su contenedor
  • right: el elemento flota a la derecha de su contenedor
  • none: el elemento no flota (se mostrará justo donde aparece en el texto). Esto es por defecto
  • heredar: el elemento hereda el valor flotante de su padre
    La propiedad clear especifica qué elementos pueden flotar al lado del elemento borrado y de qué lado.
    La propiedad clear puede tener uno de los siguientes valores:
  • none: permite elementos flotantes en ambos lados. Esto es por defecto
  • right- No se permiten elementos flotantes en el lado derecho
  • left: no se permiten elementos flotantes en el lado izquierdo
  • both: no se permiten elementos flotantes ni en el lado izquierdo ni en el derecho
  • inherit: El elemento hereda el valor claro de su padre
